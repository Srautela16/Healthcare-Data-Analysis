# Healthcare-Data-Analysis
Healthcare stroke data analysis using Python and SQL

##  Overview
This project analyzes healthcare data to identify stroke risk factors.

## Tools Used
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- SQL (MySQL)

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of key health indicators

## Insights
- Stroke risk increases with age
- High glucose level is a major factor
- BMI also influences health conditions
